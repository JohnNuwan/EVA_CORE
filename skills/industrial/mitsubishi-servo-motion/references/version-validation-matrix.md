# Mitsubishi Servo Motion - matrice de validation

A confirmer :
- Version GX Works3
- Références d'axes et servo
- Homing et limites
- Défauts remontés au PLC/HMI
- Séquence de reprise après défaut

Validation minimale :
1. Référence axe cohérente
2. Défaut servo exploitable
3. Reset/reprise documentés
4. Séquence machine découplée de la couche axe
