# Conformité biologique / organic

## Vue d'ensemble

Cette compétence sert à concevoir, auditer et maintenir la conformité biologique / organic dans les filières agroalimentaires, ingrédients et produits transformés. Elle couvre la séparation des flux, la traçabilité matière, la maîtrise des intrants autorisés, la prévention des contaminations croisées, la gestion documentaire et la préparation d'audits certificateurs.

Elle ne traite pas seulement le « label bio » comme un sujet administratif : elle le traite comme une discipline de maîtrise des flux, des campagnes, des nettoyages, des preuves et des changements de statut matière.

## Quand l'utiliser

À utiliser lorsque l'utilisateur demande de :
- Préparer un site à une certification bio/organic.
- Définir les règles de ségrégation physique ou temporelle.
- Structurer la traçabilité des lots et des intrants.
- Auditer les flux matière, stockage, nettoyage, étiquetage et preuves documentaires.
- Préparer ou corriger un audit d'organisme certificateur.
- Évaluer les risques de mélange bio / non bio ou de perte de statut.

À proscrire pour :
- Les seules exigences de sécurité alimentaire HACCP sans dimension bio : utiliser `agrifood-norms-haccp` ou `iso-22000`.
- Les environnements stériles / salles propres : utiliser `cleanroom-sterile-environments`.

## Référentiel utile

### Sources à croiser selon marché cible
- réglementations bio / organic applicables ;
- cahiers des charges certificateurs (Ecocert ou équivalent) ;
- exigences clients/distributeurs ;
- règles internes de ségrégation et de nettoyage ;
- obligations de traçabilité et d'étiquetage.

### Traduction pratique
La conformité bio ne repose pas seulement sur la matière première. Elle repose sur cinq piliers :
1. statut clair des intrants ;
2. séparation robuste des flux ;
3. nettoyage/changement de campagne maîtrisés ;
4. traçabilité documentaire sans rupture ;
5. cohérence lot / étiquette / expédition.

## Méthode de travail pas à pas

### Étape 1 — Cartographier les flux
Documenter :
- matières bio ;
- matières non bio ;
- zones de stockage ;
- lignes partagées ;
- points de transfert ;
- emballages ;
- expéditions ;
- retours / rebuts.

### Étape 2 — Définir la stratégie de ségrégation
Choisir explicitement :
- séparation physique ;
- séparation temporelle ;
- combinaison des deux ;
- règles de changement de campagne ;
- niveau de nettoyage requis ;
- règle d'étiquetage intermédiaire.

### Étape 3 — Sécuriser les intrants
Pour chaque intrant, vérifier :
- statut autorisé ;
- certificat valide ;
- cohérence fournisseur / lot / document ;
- compatibilité avec le marché visé ;
- risque de substitution ou d'erreur documentaire.

### Étape 4 — Structurer la traçabilité
Toujours pouvoir relier :
- lot entrant ;
- ordre de fabrication ;
- lot intermédiaire ;
- lot fini ;
- client / expédition ;
- justificatifs de nettoyage ;
- justificatifs de statut bio.

### Étape 5 — Préparer l'audit
Constituer un dossier d'audit avec :
- plan de flux ;
- matrice de ségrégation ;
- registre de preuves ;
- règles de changement de campagne ;
- enregistrements de nettoyage ;
- exemples de traçabilité complète.

## Matrice de décision rapide

| Sujet | Question à trancher | Critère principal |
|---|---|---|
| Ségrégation | physique ou temporelle | niveau de risque de mélange |
| Nettoyage | standard ou renforcé | sensibilité produit et historique écarts |
| Stockage | dédié ou partagé | rotation, volume et criticité |
| Étiquetage | lot par lot ou campagne | complexité logistique |
| Preuve fournisseur | minimale ou renforcée | niveau de risque marché / certificateur |

## Livrables professionnels attendus

### Minimum pour un projet sérieux
- matrice de ségrégation ;
- registre de preuves ;
- procédure de changement de campagne ;
- checklist de traçabilité ;
- dossier de conformité intrants ;
- modèle de rapport d'audit.

### Minimum pour un audit
- démonstration d'une traçabilité complète ;
- preuve de séparation des flux ;
- preuve de nettoyage entre campagnes ;
- cohérence entre lots, documents et étiquetage ;
- plan d'action sur écarts détectés.

## Cas d'usage terrain

### Atelier mixte bio / conventionnel
Points critiques :
- stockage partagé ;
- risque d'inversion lot ;
- campagnes trop serrées ;
- nettoyage insuffisamment prouvé.

### Ligne ingrédients poudres
Points critiques :
- contamination croisée résiduelle ;
- rework ;
- aspiration / transfert ;
- étiquetage intermédiaire.

### Co-packing multi-clients
Points critiques :
- statut matière ;
- séparation documentaire ;
- cohérence recette / lot / client ;
- maintien des preuves d'audit.

## Pièges Courants (Common Pitfalls)

1. Mélanger flux bio et non bio sans règles formelles.
   - Symptôme : perte de statut, produit déclassé, audit non satisfaisant.
   - Correction : formaliser séparation physique ou temporelle.

2. Avoir une traçabilité documentaire incomplète.
   - Symptôme : impossible de remonter rapidement de l'expédition à l'intrant.
   - Correction : verrouiller la chaîne lot entrant → lot fini → client.

3. Sous-estimer les exigences de changement de série / nettoyage.
   - Symptôme : risque de contamination croisée non démontré.
   - Correction : définir preuves de nettoyage et critères d'acceptation.

4. Penser que le certificat fournisseur suffit à lui seul.
   - Symptôme : incohérence entre document et flux réel.
   - Correction : recouper documents, étiquetage, stocks et lots.

5. Oublier les produits intermédiaires et reworks.
   - Symptôme : zone grise documentaire en audit.
   - Correction : tracer explicitement tout flux intermédiaire.

## Checklist de validation finale
- [ ] Les flux bio et non bio sont cartographiés.
- [ ] La stratégie de ségrégation est définie.
- [ ] Les intrants autorisés sont justifiés et archivés.
- [ ] La traçabilité lot à lot est démontrable.
- [ ] Les règles de nettoyage / changement de campagne sont documentées.
- [ ] L'étiquetage est cohérent avec le statut matière et les lots.
- [ ] Les reworks et intermédiaires sont intégrés au dispositif de traçabilité.
- [ ] Un dossier d'audit est prêt avec preuves terrain et documentaires.
