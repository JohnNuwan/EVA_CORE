# B&R Motion Control - matrice de validation

A confirmer :
- Version Automation Studio
- Axes simples / synchronisés
- Homing et références
- Contrat HMI ↔ motion
- Effet safety sur les axes
- Reprise après défaut

Validation minimale :
1. Axe prêt/non prêt cohérent
2. Homing validé
3. Défaut motion visible en maintenance
4. Séquence machine compatible avec l'état des axes
