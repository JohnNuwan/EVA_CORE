# Patterns de requêtes RAG EVA

Questions efficaces :
- "Quel standard EVA existe pour ... ?"
- "Quels documents internes couvrent ... ?"
- "Quelles conventions de nommage / architecture sont citées pour ... ?"
- "Quels écarts ou alertes les documents internes mentionnent-ils sur ... ?"

Règles de citation :
1. citer le document ou la source retrouvée ;
2. distinguer citation et interprétation ;
3. signaler si plusieurs documents se contredisent ;
4. signaler l'absence de source explicite si la réponse est partielle.
