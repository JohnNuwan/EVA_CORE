# Beckhoff TwinSAFE - matrice de validation

Points à confirmer :
- Fonctions safety actives
- Effet par axe / zone / porte
- Conditions de reset
- Effet STO / arrêt sûr
- Séquence de reprise
- Diagnostic visible côté maintenance

Validation minimale :
1. Défaut safety détecté
2. Réaction axe/machine conforme
3. Reset conditionné correctement
4. Reprise validée après retour en état sûr
