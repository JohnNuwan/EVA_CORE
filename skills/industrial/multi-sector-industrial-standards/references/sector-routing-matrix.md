# Référentiel multi-secteurs - matrice d'orientation

Pour chaque projet, qualifier séparément :
- qualité produit / procédé
- sécurité machine / procédé
- environnement
- traçabilité / data integrity
- cybersécurité OT
- exigences spécifiques secteur

Questions minimales :
1. Quel secteur pilote réellement les exigences ?
2. Quelles normes sont obligatoires vs simplement recommandées ?
3. Quels domaines doivent être traités séparément puis intégrés ?
