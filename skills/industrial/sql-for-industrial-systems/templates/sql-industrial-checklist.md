# Checklist SQL industriel

## Modèle de données
- timestamps cohérents
- qualité donnée définie
- clés lot / équipement / batch définies
- stratégie archivage définie

## Performance
- index critiques identifiés
- volumétrie estimée
- requêtes lourdes surveillées
- séparation lecture/écriture si nécessaire

## Gouvernance
- accès sécurisés
- vues/queries standardisées
- traçabilité modifications
- sauvegardes et restauration testées
