# Pilz Safety - matrice de validation

A confirmer :
- Fonctions safety couvertes
- Effet machine réel de chaque fonction
- Conditions de reset / reprise
- Zones et permissifs
- Lien PL/SIL ou norme applicable

Validation minimale :
1. Fonction déclenchée → effet machine conforme
2. Reset autorisé uniquement si préconditions remplies
3. Reprise documentée
4. Preuves terrain archivables
