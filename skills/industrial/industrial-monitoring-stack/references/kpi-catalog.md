# Catalogue KPI monitoring industriel

## KPI maintenance
- disponibilité communication équipement ;
- nombre de défauts / heure ;
- durée moyenne de blocage ;
- temps moyen de reprise ;
- MTBF ;
- MTTR ;
- nombre de resets ;
- temps depuis dernier cycle valide.

## KPI exploitation
- cadence instantanée ;
- temps de cycle moyen ;
- temps d'arrêt cumulé ;
- répartition des états ;
- pièces bonnes ;
- rebuts ;
- taux de cycles incomplets.

## KPI robot
- temps d'attente robot ;
- temps d'attente PLC / périphérie ;
- disponibilité robot ;
- nombre de défauts robot ;
- temps moyen retour Home ;
- part des arrêts imputables au robot.

## KPI énergie
- puissance instantanée ;
- énergie cumulée ;
- énergie / pièce ;
- énergie / lot ;
- dérive vs baseline ;
- charge hors production.

## KPI OEE / performance
- disponibilité ;
- performance ;
- qualité ;
- OEE / TRS ;
- micro-arrêts ;
- top pertes ;
- top défauts bloquants.

## Règles de gouvernance KPI
- définir la formule ;
- définir la source ;
- définir la fréquence de calcul ;
- définir le propriétaire métier ;
- définir les seuils et le sens d'interprétation.
