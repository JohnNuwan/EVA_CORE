# Référence approfondie B&R Automation Studio

## Points structurants
- Plateforme très cohérente pour OEM machine.
- L'intérêt maximal vient de l'intégration motion + safety + visualisation.
- La valeur projet augmente fortement avec des bibliothèques réutilisables.
- L'architecture doit rester modulaire pour famille de machines.

## Attention projet
- Ne pas laisser la HMI reconstruire la logique.
- Concevoir la safety avec le motion, pas après.
- Organiser les diagnostics pour la maintenance terrain.
