# Matrice d'évaluation design main robotique

Comparer chaque design selon :
- DoF utiles vs coût mécanique
- Complexité de commande
- Robustesse fabrication / assemblage
- Qualité de préhension sur objets cibles
- Répétabilité
- Maintenance / remplaçabilité
- Besoins capteurs / calibration

Questions minimales :
1. Quelle tâche réelle ce design améliore-t-il ?
2. Le gain de dextérité justifie-t-il la complexité ajoutée ?
3. Le prototype est-il fabricable et maintenable ?
