# Matrice formats d’échange industriels

| Format | Nature | Portabilité | Import direct outil | Cas d’usage |
|---|---|---|---|---|
| PLCopen XML | Neutre IEC | Bonne | Oui selon outil | bibliothèques, ST/FB/DUT |
| L5X | Rockwell | Faible à moyenne | Oui | AOI, UDT, tags, routines |
| XMY/XML | Schneider | Faible à moyenne | Oui | DFB, DDT, projet |
| JSON métier | Transverse | Excellente | Non direct | générateurs, contrats, OT/IT |
| CSV SCADA | Import simple | Bonne | Oui | tags, alarmes, imports massifs |
| OPC UA model | Sémantique | Bonne | Variable | interop, UNS, IA industrielle |
