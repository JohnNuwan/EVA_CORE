# Matrice de choix des langages industriels

| Famille de problème | Langage prioritaire | Alternatives | Pourquoi |
|---|---|---|---|
| Logique automate déterministe | ST / LD / FBD / SFC | GRAPH, AOI, DFB | Runtime PLC, maintenabilité terrain |
| Robotique | RAPID / TP / KRL / VAL 3 | Karel | Runtime constructeur, sécurité cellule |
| SCADA script | Jython / SQL | JavaScript, VBScript | Intégration supervision / reporting |
| Passerelle OT/IT | Python | C#, Go, Rust, Java | Vitesse de dev et bibliothèques |
| Simulation/optimisation | Python, Julia, JAX | Cython, Rust | Calcul scientifique, performance |
