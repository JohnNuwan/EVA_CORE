# Matrice Rust vs Cython

| Critère | Rust | Cython |
|---|---|---|
| Nature | langage système sûr mémoire | extension/accélération Python |
| Réutilisation code Python existant | Faible | Excellente |
| Robustesse long terme | Excellente | Bonne |
| Performance | Excellente | Très bonne |
| Courbe d’apprentissage | Haute | Moyenne |
| Cas d’usage | agents, libs, parsers, composants robustes | hot spots Python, accélération locale |
