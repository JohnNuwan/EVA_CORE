# Référence approfondie PLCnext

## Points structurants
- PLCnext est plus qu'un PLC : c'est une plateforme ouverte de contrôle + edge.
- Virtual PLC et software-defined control changent les réflexes projet.
- L'architecture doit isoler le déterministe du non déterministe.
- La gouvernance logicielle devient aussi importante que la logique automate.

## Attention projet
- Définir le comportement en cas de perte d'accès IT.
- Maîtriser les dépendances logicielles et les versions.
- Geler les interfaces OPC UA/MQTT/API dès la conception.
