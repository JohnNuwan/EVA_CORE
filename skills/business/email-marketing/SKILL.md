# Email Marketing

Compétence de niveau expert en email marketing — stratégie, automation, copywriting, délivrabilité, analyse de performance et conformité légale.

## 1. Fondamentaux de l'Email Marketing

### Types d'Emails
- **Transactional** : Confirmation de commande, réinitialisation mot de passe, notification d'expédition
- **Promotionnel** : Offres, soldes, nouveaux produits, codes promo
- **Relationnel** : Newsletters, contenu éducatif, conseils, études de cas
- **Comportemental** : Abandon panier, abandon navigation, anniversaire, inactivité
- **Lifecycle** : Welcome, onboarding, activation, upsell, réengagement, winback

### Métriques Essentielles
- **Taux d'ouverture (Open Rate)** : 15-25% est bon, > 30% excellent (moyenne sectorielle)
- **Taux de clic (CTR)** : 2-5% est bon, > 10% excellent
- **CTOR (Click-to-Open Rate)** : Clics / ouverts — qualité du contenu interne
- **Taux de rebond (Bounce Rate)** : Hard bounce (< 5%) + Soft bounce
- **Taux de désabonnement (Unsubscribe Rate)** : < 0.2% par envoi
- **Taux de conversion** : Achats / emails envoyés
- **ROI** : Retour sur investissement — moyenne de 36:1 en email marketing

### Benchmarks par Secteur
- **E-commerce** : Ouverture 15-20%, CTR 2-3%, conversion 1-2%
- **B2B** : Ouverture 20-25%, CTR 3-5%, conversion 0.5-1%
- **SaaS** : Ouverture 22-28%, CTR 3-6%, conversion 1-3%
- **Éducation** : Ouverture 20-30%, CTR 3-5%
- **Finance** : Ouverture 25-35%, CTR 3-4%
- **Voyage** : Ouverture 18-22%, CTR 2-3%

## 2. Segmentation et Ciblage

### Approches de Segmentation
- **Démographique** : Âge, sexe, localisation, langue
- **Comportementale** : Historique d'achat, navigation, engagement email, préférences
- **Lifecycle** : Prospect, lead, client actif, client dormant, churné
- **RFM** : Récence, Fréquence, Montant (score RFM : 1-5 pour chaque dimension)
- **Firmographique (B2B)** : Taille entreprise, secteur, poste, budget
- **Psychographique** : Personnalité, valeurs, centres d'intérêt

### Personnalisation Dynamique
- **Prénom** : `{{first_name}}` — basique mais efficace
- **Contenu dynamique** : Blocs affichés selon segment (produits recommandés, offres personnalisées)
- **Recommandations** : Basées sur historique d'achat, browsing, wishlist
- **Time-based** : Fuseau horaire du destinataire, anniversaire, date d'inscription
- **Product recommendations** : Moteur de recommandation (cross-sell, upsell)

## 3. Automation et Workflows

### Email Sequences Automatisées
- **Welcome Series** (3-5 emails) : Bienvenue, valeur immédiate, présentation produit, CTA
- **Onboarding Sequence** : Activation, quick wins, tutoriels
- **Abandon Panier** : H1 (1h), H2 (24h), H3 (72h avec offre)
- **Abandon Navigation** : Produits consultés, recommandations similaires
- **Post-Purchase** : Confirmation, livraison, avis, upsell
- **Winback / Réengagement** : "Vous nous manquez", offre spéciale, sondage
- **Anniversaire / Fête** : Bon d'anniversaire, offre exclusive
- **Cross-sell** : "Les clients qui ont acheté X ont aussi aimé Y"
- **Education Drip** : Séquence éducative (TTV = time to value réduit)
- **Expiry / Renewal** : Rappel expiration abonnement, offre de renouvellement

### Marketing Automation Tools
- **Mailchimp** : Small business, automation simple, templates
- **ActiveCampaign** : Automation avancée, scoring, CRM intégré
- **Brevo (Sendinblue)** : Prix abordable, SMS inclus, automation
- **HubSpot** : CRM + marketing automation, enterprise
- **ConvertKit** : Créateurs, newsletters, landing pages
- **Klaviyo** : E-commerce, analytics, segmentation avancée
- **SendGrid / Twilio** : Transactional, API, B2B, deliverability
- **MailerLite** : Simple, abordable, bon deliverability

## 4. Copywriting Email

### Objet (Subject Line)
- **Longueur** : 30-50 caractères (mobile = 25-30 caractères visibles)
- **Personnalisation** : Prénom, localisation, comportement récent
- **Curiosity gap** : Intriguer sans tout révéler
- **Urgence / Rareté** : "Plus que 24h", "Stock limité"
- **Emojis** : Test A/B — peuvent augmenter ou baisser l'ouverture selon audience
- **Pre-header** : 85-100 caractères, complète l'objet
- **Éviter les spam words** : Gratuit, argent facile, urgent, cliquer ici, offre exceptionnelle
- **A/B Test** : Toujours tester 2-3 variantes sur 10-20% de la liste

### Corps d'Email
- **Structure** : Preheader, headline, body, CTA, footer
- **Longueur** : 50-125 mots pour promotionnel, 200-400 pour newsletter
- **Formatage** : Paragraphes courts, listes, gras, espacement
- **Hiérarchie visuelle** : Headline claire, sous-titres, CTA visible (au moins 2 CTA)
- **Storytelling** : Narrative, cas client, progression logique
- **Preuve sociale** : Témoignages, chiffres, logos
- **One message per email** : Un seul objectif, un seul CTA principal

### CTA Email
- **Texte actionnel** : "Je crée mon compte", "Découvrir l'offre", "Obtenir mon guide"
- **Bouton vs Lien** : Les boutons convertissent 20-30% mieux que les liens texte
- **Placement** : Au-dessus de la ligne de flottaison + répété en fin d'email
- **Mobile** : Bouton de 44-48px minimum, contraste élevé
- **Urgence** : "Offre expire dans 24h" — compter à rebours si possible

## 5. Délivrabilité (Deliverability)

### Facteurs Clés
- **Authentication** : SPF, DKIM, DMARC — obligatoires pour Gmail et Yahoo (depuis 2024)
- **Reputation** : IP warm-up, liste qualité, faible taux de plainte (spam complaints < 0.1%)
- **Bounce handling** : Retirer immédiatement les hard bounces
- **List hygiene** : Nettoyer les inactifs tous les 6 mois, réengagement ou désabonnement
- **Engagement** : Gmail/Outlook utilisent l'engagement (ouverture, clic, déplacement en inbox) comme signal de qualité

### Configuration Technique
- **SPF (Sender Policy Framework)** : Enregistrement TXT DNS listant les IP autorisées
- **DKIM (DomainKeys Identified Mail)** : Signature cryptographique du domaine expéditeur
- **DMARC (Domain-based Message Authentication)** : Politique de traitement (none, quarantine, reject)
- **BIMI** : Logo validé à côté de l'email (nécessite DMARC = reject)
- **Reverse DNS** : Enregistrement PTR pour l'IP d'envoi
- **Feedback Loop** : S'abonner aux FBL (AOL, Yahoo, Gmail, Outlook) pour traiter les plaintes

### Warm-up
- **IP Warm-up** : Nouvelle IP = envoyer progressivement (100 → 1000 → 10k → ...)
- **Domain Warm-up** : Nouveau domaine d'envoi = comme une nouvelle IP
- **Volume** : 20% du volume total la première semaine, augmenter de 20% par semaine
- **Sent to engaged only** : Envoyer d'abord aux plus engagés (ouverts/clics récents)
- **Monitoring** : Surveiller spam rate, bounce rate, inbox placement (Glocks, Everest, Postmaster Tools)

### Outils de Monitoring
- **Google Postmaster Tools** : Réputation, taux de spam, authentification (domaines Gmail)
- **Microsoft SNDS** : Réputation Outlook/Hotmail
- **Glocks / Everest / Mailgun** : Inbox placement tests
- **MXToolbox** : Vérification SPF/DKIM/DMARC
- **SpamAssassin** : Score spam avant envoi

## 6. Conception et Templates

### Responsive Design
- **Layout** : Single-column, largeur max 600px
- **Font** : Web-safe (Arial, Helvetica, Georgia, Verdana), taille 14-16px body
- **Images** : Alt text obligatoire, taille < 1MB, WebP supporté (fallback JPG/PNG)
- **CTA** : Bouton avec padding, pas d'image CTA (images bloquées par défaut)
- **Dark mode** : Tester les couleurs, bords blancs pour les boutons
- **Preview** : Litmus, Email on Acid, Sinch Email on Acid

### Templates
- **Modular templates** : Blocs réutilisables (header, hero, content, footer)
- **Plain text** : Alternative texte pour les clients email sans HTML
- **AMP for Email** : Emails interactifs (formulaires, carrousels, produits) — supporté par Gmail, Outlook, Yahoo
- **Accessibilité** : Ratio de contraste, focus states, texte alternatif, landmarks

## 7. Législation et Conformité

### RGPD (UE)
- **Consentement explicite** : Opt-in, case non cochée par défaut (pas de consentement pré-coché)
- **Double opt-in** : Email de confirmation après inscription (recommandé fortement)
- **Droit d'accès** : L'utilisateur peut demander toutes ses données
- **Droit d'oubli** : Supprimer les données sur demande
- **Base légale** : Consentement ou intérêt légitime
- **DPO** : Délégué à la protection des données (obligatoire si volume important)
- **Registre** : Tenir un registre des traitements

### CAN-SPAM (USA)
- **Opt-out** : Option de désabonnement visible et fonctionnelle
- **Header** : Expéditeur et sujet non trompeurs
- **Physical address** : Adresse postale valide
- **Délai** : Traiter les désabonnements sous 10 jours ouvrés

### CASL (Canada)
- **Consentement explicite** : Opt-in, pas de présomption
- **Désabonnement** : Liens de désabonnement faciles d'accès
- **Conservation** : Garder la preuve du consentement

### Anti-spam
- **CAN-SPAM, CASL, Loi pour la Confiance dans l'Économie Numérique (France)**
- **Fréquence** : Pas plus d'un email par jour pour les newsletters (sauf consentement explicite)
- **Respect des désabonnements** : Traitement immédiat

## 8. A/B Testing Email

### Éléments à Tester
- **Objet** : Ton, longueur, personnalisation, emojis, mots-clés
- **Pre-header** : Complément à l'objet
- **CTA** : Texte, couleur, taille, position
- **Contenu** : Long vs court, storytelling vs liste
- **Design** : Single column vs multi, images vs pas d'images
- **Heure d'envoi** : Matin vs après-midi, jour de la semaine
- **Expéditeur** : Nom vs nom + prénom, marque vs personne

### Protocole de Test
1. **Segment test** : 10-20% de la liste
2. **Taille d'échantillon** : Minimum 1000 contacts par variante
3. **Durée** : 24-48h (selon secteur)
4. **Significativité** : 95% de confiance minimum
5. **Gagnant** : Envoyer la variante gagnante au reste de la liste

## 9. Pièges et Antipatterns

- **Acheter des listes** : Illégal (RGPD) et destructeur pour la réputation
- **Envoyer trop souvent** : Fatigue des abonnés, augmentation des désabonnements et plaintes spam
- **Pas de mobile optimization** : 50-70% des emails sont ouverts sur mobile
- **Images bloquées** : Ne pas mettre d'info critique uniquement dans des images
- **Subject line trompeuse** : Augmente l'ouverture mais détruit la confiance à long terme
- **Ignorer les non-engageants** : Nettoyer les inactifs régulièrement
- **Pas de plain text version** : Certains clients email bloquent le HTML
- **Trop de CTA** : Dilue l'attention, réduit le taux de clic
- **Pas de test avant envoi** : Liens cassés, design cassé, erreurs de merge tags
- **Ne pas respecter RGPD** : Amendes jusqu'à 20M€ ou 4% du CA annuel mondial
