# Publicité Payante (PPC/SEA)

Compétence de niveau expert en publicité payante — Google Ads, Social Ads, programmatique, stratégie d'enchères, optimisation de campagnes et mesure du ROI.

## 1. Fondamentaux PPC

### Modèles de Facturation
- **CPC (Cost Per Click)** : Paiement par clic — modèle standard Search, Display, Social
- **CPM (Cost Per Mille)** : Paiement pour 1000 impressions — branding, awareness
- **CPA (Cost Per Acquisition)** : Paiement par conversion — réseaux d'affiliation
- **CPV (Cost Per View)** : Paiement par vue (YouTube, vidéo)
- **CPL (Cost Per Lead)** : Paiement par lead qualifié
- **Flat Rate** : Tarif fixe par période ou emplacement

### Métriques Essentielles
- **CTR (Click-Through Rate)** : Clics / Impressions — quality score
- **CPC moyen** : Coût moyen par clic
- **CVR (Conversion Rate)** : Conversions / Clics
- **CPA / CPLead** : Coût par acquisition ou lead
- **ROAS (Return on Ad Spend)** : Revenu / Dépenses publicitaires
- **ROI** : (Revenu - Coût) / Coût × 100
- **Quality Score** : Google Ads 1-10 — basé sur CTR, pertinence, landing page
- **Impression Share** : % d'impressions obtenues vs disponibles

### Structure de Compte
- **Campaign** : Objectif, budget, enchères, ciblage
- **Ad Group** : Thème, mots-clés, audience
- **Ad** : Copies, créatives, extensions
- **Keyword** : Match types (exact, phrase, broad, broad modified — simplifié en 2024)
- **Budget** : Quotidien, partagé, évité si possible (budgets de campagne séparés)

## 2. Google Ads

### Types de Campagnes Google
- **Search** : Mots-clés, text ads, extensions — intention de recherche
- **Display** : Image, responsive display ads, remarketing — audience ciblage
- **Shopping** : Product listing ads (PLA), Merchant Center — e-commerce
- **Video (YouTube)** : In-stream, bumper, discovery — vidéo
- **Performance Max** : IA Google, tous les canaux Search + Display + YouTube + Discovery + Gmail
- **App** : Universal App Campaign — promotion d'applications mobiles
- **Demand Gen** : YouTube + Discover + Gmail — formats visuels inspirés de Social
- **Local** : Annonces dans Google Maps, profil d'établissement

### Search Campaigns — Mots-Clés
- **Match Types** (simplifiés 2024) :
  - **Exact** : `[assurance auto]` — mot exact ou variantes proches
  - **Phrase** : `"assurance auto"` — ordre des mots respecté
  - **Broad** : `assurance auto` — mots-clés proches, synonymes, requêtes liées
- **Negative Keywords** : Mots-clés à exclure (ex: "gratuit", "bricolage")
- **Keyword Research** : Google Keyword Planner, Search Terms Report, concurrents
- **SKAG vs STAG** : Single Keyword Ad Group vs Single Theme Ad Group (STAG recommandé)
- **Match Type Strategy** : Exact pour contrôle, broad pour discovery (avec smart bidding)

### Quality Score
1. **Expected CTR** : Probabilité de clic sur l'annonce
2. **Ad Relevance** : Pertinence du texte d'annonce par rapport au mot-clé
3. **Landing Page Experience** : Qualité, pertinence, vitesse de la page de destination
- **Impact** : QS élevé = meilleur rang, CPC plus bas

### Google Ads Extensions
- **Sitelink** : Liens supplémentaires vers des pages spécifiques
- **Callout** : Texte descriptif (ex: "Livraison gratuite", "Support 24/7")
- **Structured Snippet** : Liste (ex: "Services : Audit, Conseil, Formation")
- **Call** : Numéro de téléphone cliquable
- **Location** : Adresse, carte, distance
- **Price** : Prix des produits/services
- **Promotion** : Offre spéciale, code promo
- **Image** : Image dans les annonces Search (bêta 2024)
- **App** : Lien vers application mobile

### Performance Max
- **Objectif** : Maximiser les conversions sur tous les canaux Google
- **Assets** : 5+ images, 5+ titres, 5+ descriptions, logo, vidéo
- **Audience signals** : Indices d'audience (remarque : signaux, pas targeting)
- **Data exclusion** : Exclure les marques, les mots-clés négatifs
- **Reporting** : Insights, asset performance, reach planning
- **Avantages** : IA Google optimise automatiquement le mix de canaux
- **Limites** : Moins de contrôle ("boîte noire"), cannibalise les autres campagnes

### Google Display Network (GDN)
- **Audience targeting** : Affinité, marché, remarketing, segments personnalisés, in-market
- **Contextual targeting** : Mots-clés, sujets, placements
- **Demographic targeting** : Âge, sexe, parental status
- **Remarketing** : Site visitors, YouTube viewers, listes clients (Customer Match)
- **Responsive Display Ads** : Images + titres + descriptions = combinés automatiquement
- **Placements exclus** : Éviter les sites de faible qualité

### Google Shopping
- **Merchant Center** : Flux produit (titre, description, prix, disponibilité, image, GTIN)
- **Product Feed Optimization** : Titres riches (marque + type + attributs), descriptions optimisées
- **Promotions** : Réductions, livraison offerte
- **Free listings** : Listings gratuits dans Shopping
- **Surfaces across Google** : Images produit apparaissant dans Google Images, Lens, etc.
- **Shopping campaigns** : Standard, Smart Shopping (remplacé par PMax), Performance Max

## 3. Social Ads

### Facebook & Instagram Ads
- **Campaign Objectives** : Awareness (Brand), Traffic, Engagement, Leads, Sales
- **Ad Formats** : Image, Video, Stories, Reels, Carousel, Collection, Messenger
- **Targeting** :
  - **Core** : Démographie, intérêts, comportements
  - **Custom** : Pixel visitors, email list (Customer Match), app users
  - **Lookalike** : 1-10% — basé sur Custom Audience
- **Meta Pixel** : Événements standards (Purchase, Lead, AddToCart), événements personnalisés
- **CAPI (Conversions API)** : Server-side tracking (contourne les bloqueurs de cookies)
- **Advantage+** : Optimisation IA des audiences et créatives
- **Retargeting** : 7-30 jours, pages spécifiques (panier, produit)
- **Budget** : CBO (Campaign Budget Optimization), advantage campaign budget
- **Ad fatigue** : Fréquence > 3-4 → renouveler les créatives
- **Creative testing** : Test and learn, Dynamic Creative, Advantage+ Creative

### LinkedIn Ads
- **Campaign Types** : Sponsored Content, Message Ads, Text Ads, Dynamic Ads, Carousel
- **Targeting** : Job title, function, seniority, industry, company size, skills, groups, education
- **Matched Audiences** : Website retargeting, contact list, account targeting (ABM)
- **Lead Gen Forms** : Formulaire pré-rempli LinkedIn (haut taux de conversion)
- **Insight Tag** : LinkedIn pixel, conversion tracking, website audiences
- **CPC/CPM** : Plus élevé que Google/Facebook (5-10€ CPC LinkedIn vs 0.5-1€ Facebook B2C)
- **Best for** : B2B, recrutement, contenus premium, formation professionnelle

### TikTok Ads
- **Formats** : In-feed, Spark Ads, TopView, Branded Hashtag Challenge, Branded Effect
- **Targeting** : Interest, behavior, demography, custom audience (app activity)
- **TikTok Pixel** : Événements de conversion (standard), retargeting
- **Creative** : Contenu UGC-style, authentique, tendance, pas trop produit
- **Budget** : Minimum 50€/jour par campagne
- **Benchmark** : CPM ~3-8€, CPC ~0.2-1€, CTR ~1-3%

## 4. Stratégie d'Enchères

### Google Ads Bidding
- **Manual CPC** : Contrôle total, basé sur l'expérience
- **Enhanced CPC (ECPC)** : Ajustement semi-automatique
- **Target CPA** : Google optimise pour un CPA cible
- **Target ROAS** : Google optimise pour un ROAS cible (recommandé si données suffisantes)
- **Maximize Conversions** : Budget fixe, optimise pour le max de conversions
- **Maximize Conversion Value** : Budget fixe, optimise pour la valeur de conversion
- **Target Impression Share** : Pour la visibilité
- **Smart Bidding** : Machine learning de Google (CPA, ROAS, Maximize)

### Facebook Bidding
- **Lowest Cost (sans limite)** : Facebook optimise pour le coût le plus bas
- **Lowest Cost (avec limite de coût)** : Plancher de CPA
- **Cost Cap** : Maximum de CPA ou ROAS minimum
- **Bid Cap** : Enchère manuelle maximale (avancé, risque de sous-dépense)
- **Minimum ROAS** : ROAS cible (Minimum return on ad spend)

### Enchères Avancées
- **A/B Testing** : Comparer stratégies d'enchères
- **Portfolio bid strategies** : Stratégie partagée entre plusieurs campagnes
- **Seasonality adjustments** : Ajustements saisonniers (Black Friday, Noël)
- **Conversion delay** : Prendre en compte le délai de conversion (modèle d'attribution)
- **Data threshold** : Minimum 20-30 conversions par semaine pour Smart Bidding

## 5. Tracking et Attribution

### Mise en Place du Tracking
- **Conversion Tracking** : Google Ads, Facebook Pixel, LinkedIn Insight Tag, TikTok Pixel
- **Server-side Tracking** : Facebook CAPI, Google Tag Manager Server-side, sGTM
- **UTM Parameters** : `?utm_source=google&utm_medium=cpc&utm_campaign=nom&utm_content=ad&utm_term=keyword`
- **Google Analytics 4** : Événements de conversion importés dans Google Ads
- **Offline Conversions** : Import des ventes hors ligne via GCLID, FBCLID
- **Call Tracking** : Numbers dynamiques, CallRail, Google Forwarding Numbers

### Modèles d'Attribution
- **Last Click** : Attribué au dernier clic — par défaut
- **First Click** : Attribué au premier clic
- **Linear** : Distribution égale
- **Time Decay** : Plus l'interaction est récente, plus elle est pondérée
- **Position Based (U-Shaped)** : 40% first, 40% last
- **Data-Driven** : Modèle algorithmique (Google Ads, Facebook — si données suffisantes)
- **Cross-device** : Attribution multi-appareil (Google Signals, Facebook)

## 6. Optimisation de Campagne

### Search Ads Optimization
- **Search Terms Report** : Ajouter des mots-clés positifs, négatifs, exact match
- **Ad rotation** : Tester titres, descriptions, CTA, extensions
- **Landing page optimization** : Pertinence, vitesse, conversion (CRO)
- **Auction Insights** : Part d'impressions, position moyenne vs concurrents
- **Quality Score** : Amélioration des annonces pour augmenter le QS

### Display Optimization
- **Placement exclusions** : Sites à faible performance
- **Frequency capping** : Limiter les impressions par utilisateur
- **Audience refinement** : Affiner les audiences en fonction du ROAS
- **Creative rotation** : Actualiser les bannières (fatigue : 3-4 semaines)
- **View-through conversions** : Conversions sans clic (impression uniquement)

### Social Ads Optimization
- **Ad fatigue** : Surveiller frequency, CTR, CPM → renouveler si fréquence > 3-4
- **Creative testing** : Minimum 3-4 créatives par ad set
- **Audience refinement** : Retargeting vs prospection — budgets séparés
- **Ad schedule** : Jours et heures à plus fort ROAS
- **Device optimization** : Mobile vs desktop — ajuster les bids

### Budget Management
- **PACED Framework** : Pacing, Allocation, Changes, Efficiency, Data
- **Budget pacing** : Répartition uniforme vs accéléré (delivery method)
- **Dayparting** : Ajuster les enchères selon l'heure
- **Geographic optimization** : Régions à plus fort ROI
- **Device bid adjustment** : Mobile +X%, Desktop +Y%
- **Seasonal adjustments** : Augmenter les budgets pour les périodes clés

## 7. ROAS et Reporting

### Calcul du ROAS
- **ROAS** = Revenu attribué / Dépenses publicitaires
- **ROAS Cible** : Basé sur la marge — ex: si 25% de marge, ROAS minimum = 4 (×4)
- **Blended ROAS** : ROAS global tous canaux confondus
- **Channel ROAS** : ROAS par canal (Google vs Facebook vs LinkedIn)
- **Campaign ROAS** : ROAS par campagne
- **Segment ROAS** : ROAS par appareil, localisation, audience

### Rapports et Dashboards
- **Google Ads** : Rapports natifs, segments, automatisation
- **Google Looker Studio** : Tableaux de bord personnalisés multi-sources
- **Google Sheets + Apps Script** : Rapports automatisés
- **Supermetrics** : Data pipeline vers Sheets, Looker, Data Studio
- **Triple Whale / Northbeam / Rockerbox** : Attribution avancée multi-canal
- **Rapports automatisés** : Emails hebdomadaires aux parties prenantes

## 8. Pièges et Antipatterns

- **Lancer sans tracking** : Impossible d'optimiser sans conversion tracking
- **Smart Bidding trop tôt** : Besoin de données historiques de conversion (30+ conversions/30 jours)
- **Ignorer les negative keywords** : Gaspillage sur des requêtes non pertinentes
- **Budget trop petit** : Budget minimum = 10-15× CPC pour collecter des données
- **Créatives non actualisées** : Ad fatigue → baisse du CTR, augmentation du CPM et CPC
- **Mobile non optimisé** : 60-80% du trafic est mobile
- **Remarketing excessif** : Fréquence > 5 → irritation des utilisateurs
- **Non prise en compte du délai de conversion** : Conversion delay — attendre 7-14 jours avant d'optimiser
- **Budget trop large (PMax)** : Performance Max sans données de conversion historiques
- **Attribution incorrecte** : Last-click donne tout le crédit au dernier touchpoint
- **Pas de test A/B** : Optimisation basée sur l'intuition, pas sur les données
- **Dépendance à un seul canal** : Diversifier les canaux d'acquisition
