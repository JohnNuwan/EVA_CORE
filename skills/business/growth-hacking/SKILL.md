# Growth Hacking

Compétence de niveau expert en Growth Hacking — combinaison de marketing, data science et ingénierie pour une croissance rapide et mesurable à faible coût.

## 1. Framework AARRR (Pirate Metrics)

### Acquisition — Attirer les utilisateurs
- **Canaux payants** : Facebook Ads, Google Ads, TikTok Ads, LinkedIn Ads, Apple Search Ads
- **Canaux organiques** : SEO, content marketing, médias sociaux, ASO (App Store Optimization)
- **Canaux viraux** : Parrainage, partage, boucles virales, embed widgets
- **Canaux de growth** : Growth loops, product-led growth, community-led growth
- **Métriques** : CAC (Customer Acquisition Cost), CPA (Cost Per Acquisition), CTR, CPM, coût par lead

### Activation — Première expérience "Aha!"
- **Time to Value (TTV)** : Temps entre l'inscription et la première valeur perçue — idéal < 5 minutes
- **Onboarding** : Wizard, checklist, échantillon gratuit, démo interactive, tutoriel progressif
- **Quick wins** : Premiers succès immédiats (compléter un profil, créer un premier projet)
- **Métriques** : Taux d'activation, % d'utilisateurs ayant atteint le "Aha moment"

### Rétention — Fidéliser les utilisateurs
- **Cohort analysis** : Rétention J1, J7, J30 — courbe de rétention
- **Engagement loops** : Notifications push, emails, in-app messages, gamification
- **Habit formation** : Boucles d'habitude (Trigger → Action → Reward → Investment)
- **Churn prediction** : Modèles ML pour prédire le désabonnement
- **Métriques** : Taux de rétention, churn rate, DAU/MAU (Daily/Monthly Active Users), stickiness (DAU/MAU)

### Revenu — Monétiser
- **Modèles de pricing** : Freemium, abonnement, à l'usage, transactionnel, publicité
- **Upsell / Cross-sell** : Vente incitative, ventes croisées, bundles
- **Pricing psychology** : Encadrement (anchoring), leurre (decoy), gratuit + premium
- **Métriques** : ARPU (Average Revenue Per User), ARPPU, LTV (Lifetime Value), MRR/ARR

### Référence — Bouche-à-oreille viral
- **Viral coefficient (K)** : K = nombre d'invitations × taux de conversion — K > 1 = croissance virale
- **Cycle viral** : Temps entre l'envoi d'une invitation et l'activation du nouvel utilisateur
- **Programmes de parrainage** : Dropbox (500 Mo par ami), PayPal (10 $), Uber (crédit course)
- **Métriques** : Taux de parrainage, viral coefficient, cycle viral

## 2. Expérimentation et Growth Loops

### Boucles de Croissance (Growth Loops)
- **Boucle virale** : Utilisateur → produit → partage → nouvel utilisateur (ex: Hotmail signature email)
- **Boucle de contenu** : Création de contenu → SEO → trafic → conversion → nouveau contenu
- **Boucle de vente** : Lead → essai → achat → upsell → expansion
- **Boucle communauté** : Membre → contenu communautaire → attractivité → nouveau membre
- **Boucle de données** : Utilisateur → données → meilleur produit → plus d'utilisateurs (ex: Waze)

### Processus d'Expérimentation
1. **Hypothèse** : Si [changement], alors [résultat], parce que [raison]
2. **Priorisation ICE** : Impact × Confidence × Ease (score)
3. **Design** : Test A/B, split test, test multivarié
4. **Exécution** : Implémenter, suivre, itérer
5. **Analyse** : Significativité statistique (p < 0.05), power analysis, lift
6. **Apprentissage** : Documenter, scaler (si gagnant), tuer (si perdant)

### Outils d'Expérimentation
- **A/B Testing** : Google Optimize, Optimizely, VWO, AB Tasty
- **Feature flags** : LaunchDarkly, Split.io, Flagsmith
- **Heatmaps** : Hotjar, Crazy Egg, Microsoft Clarity
- **Session recording** : FullStory, Hotjar, LogRocket
- **Surveys** : Typeform, SurveyMonkey, Intercom

## 3. Growth Hacking par Canal

### Product-Led Growth (PLG)
- **Freemium** : Version gratuite limitée (Slack, Dropbox, Canva)
- **Free trial** : Essai gratuit limité dans le temps (14-30 jours)
- **Self-serve** : Inscription et activation sans intervention humaine
- **Virality intégrée** : Partage obligatoire pour débloquer une fonctionnalité
- **Network effects** : Plus d'utilisateurs = plus de valeur (airdrop effect)

### Community-Led Growth
- **Communauté propriétaire** : Slack/Discord/Telegram privé
- **User-generated content** : Avis, tutoriels, templates créés par les utilisateurs
- **Champions program** : Ambassadeurs, early adopters, power users
- **Co-création** : Impliquer la communauté dans le développement produit

### Content-Led Growth
- **SEO Content** : Blog, guides, landing pages longue traîne
- **Video marketing** : YouTube, TikTok, Reels — tutoriels, démos, reviews
- **Webinars** : Webinaires en direct, formations gratuites, masterclass
- **Podcasts** : Interviews, études de cas, industry insights
- **Tools gratuits** : Outils en ligne gratuits (calculators, checkers, generators) qui génèrent leads

### Channel-Led Growth
- **Affiliation** : Programmes d'affiliation (ShareASale, Impact, Refersion)
- **Partnerships** : Intégrations API, plateformes, resellers
- **Marketplace** : Distribution via AppSumo, Product Hunt, G2, Capterra
- **Influencer** : Micro-influenceurs (10k-100k followers), nano-influenceurs

## 5. Growth Metrics avancées

### Metrics Composées
- **CAC Payback** : Temps pour récupérer le coût d'acquisition (CAC/ARPU mensuel)
- **LTV/CAC Ratio** : > 3 = sain, < 1 = non rentable
- **Quick Ratio** : (Nouveau MRR + Expansion MRR) / Churned MRR — > 1 = croissance
- **Net Revenue Retention (NRR)** : Revenu récurrent net (expansion - churn), > 120% = excellent
- **Magic Number** : (Croissance ARR trimestrielle × 4) / Dépenses sales & marketing du trimestre précédent

### North Star Metric
- **Définition** : La métrique unique qui corrèle le mieux avec la valeur à long terme
- **Exemples** : Airbnb = nuits réservées, Spotify = heures d'écoute, Facebook = DAU
- **Caractéristiques** : Mesurable, prédictive de la valeur, actionnable, compréhensible par tous

## 6. Growth Hacking Playbooks

### Playbook Viral
1. **Embed strategy** : Widget embeddable avec branding (Hotjar, Typeform)
2. **Signature email** : Lien de parrainage dans la signature (Hotmail, Mailchimp)
3. **Share with friends** : Partage obligatoire pour débloquer un accès (Dropbox)
4. **Invite-only** : Accès sur invitation, liste d'attente (Clubhouse, Gmail)
5. **Cooperative growth** : Partenaires qui partagent la base d'utilisateurs (Uber × Spotify)

### Playbook PLG
1. **Self-serve onboarding** : Créer un compte et utiliser le produit sans assistance humaine
2. **Time-to-value minimal** : Réduire à < 2 minutes (ex: Loom, Calendly)
3. **Usage-based triggers** : Email automatique après une action clé
4. **Expansion revenue** : Upsell automatique basé sur l'utilisation
5. **API growth** : Intégrations API qui attirent des utilisateurs (Stripe, Twilio)

## 7. Pièges et Antipatterns

- **Vanity metrics** : Métriques qui font bonne impression mais n'impactent pas le business (pages vues, téléchargements, followers)
- **Growth sans rétention** : Acquérir massivement mais ne pas retenir → seau percé
- **Saturation des canaux** : Dépendance excessive à un seul canal d'acquisition
- **Spam viral** : Forcer le partage sans valeur → mauvais signal et pénalité plateforme
- **Ignorer la segmentation** : Traiter tous les utilisateurs de la même manière
- **Tests non significatifs** : Arrêter un test trop tôt ou sans puissance statistique
- **Oublier le produit** : Growth sans amélioration du produit → désabonnement
