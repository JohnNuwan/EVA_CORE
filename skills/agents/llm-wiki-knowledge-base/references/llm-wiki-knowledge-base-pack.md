## Résumé Original de l'Article
LLM Wiki 2.0 pattern defines a personal knowledge management system for AI agents using interconnected Markdown pages. Instead of passive flat vector retrieval, the agent extracts entities into a structured wiki guided by Purpose and Schema files, preventing information overload (anti-FOMO) through automated clustering and distillation.
