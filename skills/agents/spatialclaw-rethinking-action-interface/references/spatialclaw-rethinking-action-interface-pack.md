# Référence Scientifique : SpatialClaw: Rethinking Action Interface for Agentic Spatial Reasoning

*   **Auteurs** : Seokju Cho, Ryo Hachiuma, Abhishek Badki, Hang Su
*   **Identifiant arXiv** : 2606.13673
*   **Lien arXiv** : https://arxiv.org/abs/2606.13673

---

## Résumé Original de l'Article
Existing spatial agents struggle with limited flexibility using single-pass code execution or structured tool-calls. We introduce SpatialClaw, a training-free framework that uses code as the action interface. It employs a stateful Python kernel pre-loaded with perception and geometry primitives, allowing a VLM-backed agent to write executable code step-by-step to flexibly compose operations.
