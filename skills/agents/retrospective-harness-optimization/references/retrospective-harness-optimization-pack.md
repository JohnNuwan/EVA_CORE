# Référence Scientifique : Retrospective Harness Optimization: Improving LLM Agents via Self-Preference over Trajectory Rollouts

*   **Auteurs** : Wenbo Pan, Shujie Liu, Chin-Yew Lin, Jingying Zeng
*   **Identifiant arXiv** : 2606.05922
*   **Lien arXiv** : https://arxiv.org/abs/2606.05922

---

## Résumé Original de l'Article
An agent's performance depends heavily on its harness (instructions, skills, and tools). We introduce Retrospective Harness Optimization (RHO), a method to improve LLM agents in the dark by mining past trajectories. RHO extracts difficult failure cases as a coreset, re-solves them in parallel with mutated harnesses, and uses self-preference to select and update the optimal harness.
