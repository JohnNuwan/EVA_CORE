# Référence Scientifique : When LLMs Read Tables Carelessly: Measuring and Reducing Data Referencing Errors

*   **Auteurs** : Yuqing Yang, Qi Zhu, Zhen Han, Boran Han
*   **Identifiant arXiv** : 2606.32029
*   **Lien arXiv** : https://arxiv.org/abs/2606.32029

---

## Résumé Original de l'Article
LLMs tend to make Data Referencing Errors (DREs)—such as incorrectly citing or omitting values—when processing tabular data, even when the models appear to understand the table structure. We systematically evaluate DREs and show that incorporating data referencing as a critic can improve answer accuracy by up to 12% using critic-based filtering and rejection sampling.
