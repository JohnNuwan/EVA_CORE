*   **Identifiant arXiv** : 2605.20682
*   **Lien arXiv** : https://arxiv.org/abs/2605.20682

## Résumé Original de l'Article
Existing multimodal models (MLLMs) face challenges in open-vocabulary industrial anomaly detection (IAD) due to domain-misaligned reasoning. We introduce IndusAgent, a tool-augmented agentic framework that resolves visual ambiguities using dynamic region cropping, high-frequency visual feature enhancement, and prior retrieval, guided by a gated reinforcement learning objective.
