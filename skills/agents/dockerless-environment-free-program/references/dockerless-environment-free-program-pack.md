# Référence Scientifique : Dockerless: Environment-Free Program Verifier for Coding Agents

*   **Auteurs** : Wenhao Zeng, Yuling Shi, Xiaodong Gu, Chao Hu, Chaofan Wang
*   **Identifiant arXiv** : 2606.28436
*   **Lien arXiv** : https://arxiv.org/abs/2606.28436

---

## Résumé Original de l'Article
Traditional program verification for coding agents typically requires execution-based testing within isolated environments (like Docker containers), which is resource-intensive, slow, and often infeasible for repositories without reproducible environments or test suites. We introduce Dockerless, an environment-free, agentic patch verifier. Instead of running code, it evaluates generated patches by actively exploring the repository and gathering sémantic evidence to judge correctness.
