# Référence Scientifique : Memory is Reconstructed, Not Retrieved: Graph Memory for LLM Agents

*   **Auteurs** : Shuo Ji, Yibo Li, Bryan Hooi
*   **Identifiant arXiv** : 2606.06036
*   **Lien arXiv** : https://arxiv.org/abs/2606.06036

---

## Résumé Original de l'Article
Existing retrieve-then-reason frameworks for LLM agents fail to adapt to long interaction histories. We introduce MRAgent, a framework that uses an associative Cue-Tag-Content graph to represent memory. Instead of passive retrieval, it employs an active reconstruction mechanism that integrates LLM reasoning directly into memory access, allowing the agent to dynamically explore and refine retrieval paths.
