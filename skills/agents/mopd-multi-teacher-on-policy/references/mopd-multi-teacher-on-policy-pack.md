# Référence Scientifique : MOPD: Multi-Teacher On-Policy Distillation for Capability Integration in LLM Post-Training

*   **Auteurs** : Wenhan Ma, Jianyu Wei, Liang Zhao, Hailin Zhang
*   **Identifiant arXiv** : 2606.30406
*   **Lien arXiv** : https://arxiv.org/abs/2606.30406

---

## Résumé Original de l'Article
Integrating specialized reinforcement learning teachers into a student LLM during post-training is challenging. We introduce MOPD (Multi-Teacher On-Policy Distillation) for capability integration. MOPD performs specialized RL per domain and distill these teachers using the student's on-policy rollouts, avoiding exposure bias.
