## Lecture recommandée pour les demandes futures

### Si l'utilisateur demande une montée en compétence générale
Répondre dans l'ordre :
1. familles déjà couvertes ;
2. cas d'usage par constructeur ;
3. protocoles transverses ;
4. zones restant à approfondir.

### Si l'utilisateur demande une standardisation multi-constructeurs
Comparer systématiquement :
- environnement d'ingénierie ;
- modèle de données ;
- bus terrain ;
- exposition SCADA/MES ;
- capacités de motion, batch, safety, redondance.

## Heuristique d'honnêteté
Ne pas dire "tous les automates de l'industrie" au sens absolu.
Préférer :
- "les principales plateformes majeures déjà couvertes sont..."
- "les familles restantes à approfondir sont..."
