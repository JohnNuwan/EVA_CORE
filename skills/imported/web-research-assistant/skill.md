# Web Research Assistant Skill

Effectue une recherche web structurée avec extraction IA et synthèse.

## Usage

Combine `ai_scrape`, `web_search` et `scrape_sitemap` pour une recherche complète.

## Procédure

1. **Requête** — formulation de la question de recherche
2. **Découverte** — `web_search` pour trouver les sources + `scrape_sitemap` pour explorer
3. **Extraction** — `ai_scrape` ou `extract_structured` pour extraire le contenu pertinent
4. **Synthèse** — compilation des résultats en rapport structuré
5. **Citation** — chaque source citée avec URL
