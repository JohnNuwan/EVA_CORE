# Analyse des Médias Sociaux Computationnelle

## Présentation

Compétence niveau expert en analyse des médias sociaux par méthodes computationnelles et IA. Couvre les APIs plateformes, le social listening, l'analyse d'influence et de viralité, le community management, et le social selling.

---

## 1. APIs Plateformes

### Accès aux Données

- **Twitter/X API v2** : tweets, search, streams, engagement metrics, compliance
  - OAuth 2.0, rate limits, filtered stream, recent/full-archive search
- **Reddit Pushshift/API** : submissions, comments, subreddits
  - Pushshift pour données historiques, Reddit API pour temps réel
- **TikTok Research API** : vidéos, utilisateurs, commentaires (accès académique)
- **Instagram Graph API** : médias, insights, mentions, hashtags
  - Instagram Basic Display API pour données publiques
- **LinkedIn API** : posts, company pages, engagement analytics
  - LinkedIn Marketing API pour publicité
- **YouTube Data API v3** : vidéos, playlists, commentaires, analytics
- **Facebook Graph API** : posts, pages, insights, mentions
  - Page Public Content Access

### Bonnes Pratiques

- **Scraping Éthique** : respect des ToS, robots.txt, attribution
- **Rate Limits** : gestion des limites de requêtes (backoff exponentiel, throttling)
- **Quotas** : allocation de quota APIs (coût par endpoint)
- **Data Storage** : conformité RGPD pour le stockage des données sociales

---

## 2. Social Listening

### Surveillance de Marque

- **Brand Monitoring** : suivi des mentions de marque en temps réel
- **Sentiment Analysis** : analyse de sentiment (positif, négatif, neutre, mixte)
- **Volume Trends** : tendances de volume de mentions
- **Share of Voice** : part de voix par rapport aux concurrents
- **Topic Clustering** : regroupement thématique des conversations (LDA, BERTopic)
- **Conversation Mining** : extraction d'insights des conversations
- **Crisis Detection Alert** : alertes de détection de crise en temps réel

### Analyse de Contenu

- Hashtag analysis, keyword trends
- Image recognition pour logos/produits
- Geolocation des conversations
- Cross-platform integration

---

## 3. Influence et Viralité

### Identification d'Influenceurs

- **Influencer Identification** : découverte d'influenceurs par pertinence et portée
- **Klout Score** : mesure historique d'influence sociale
- **Reach Prediction** : prédiction de portée d'une publication
- **Engagement Forecasting** : prévision d'engagement (likes, comments, shares)
- **Viral Coefficient** : coefficient viral (partages par vue × conversion)
- **Content Propagation** : modélisation de propagation de contenu
- **Network Effects** : analyse des effets de réseau

### Propagation et Graphes

- **Graph Influence Spread** : propagation d'influence sur le graphe social
- **Cascade Prediction** : prédiction de cascades virales
- **Community Detection** : détection de communautés (Louvain, Girvan-Newman)
- **Bot Detection** : détection de bots sociaux (ML features : ratio followers/following, timing, content)
- **Influence Maximization** : sélection de nœuds maximisant la propagation

## 4. Community Management

### Gestion de Communauté

- **Engagement Metrics** : métriques d'engagement (ER, interactions par post)
- **Response Time Optimization** : optimisation des temps de réponse
- **Sentiment-Based Triaging** : triage par urgence sentimentale
- **Auto-Responder LLM** : réponses automatiques par LLM
- **Toxicity Detection** : détection de toxicité (Perspective API, modèles de classification)
- **Moderation** : modération automatisée des contenus

### Rapports et Gestion de Crise

- **Report Generation** : génération automatisée de rapports
- **Crisis Management** : gestion de crise — détection précoce, escalation, réponse
- **Crisis Communication** : communication de crise par IA
- **Employee Advocacy** : programmes de défense des employés sur les réseaux

## 5. Social Selling

### Vente Sociale

- **Lead Generation Social** : génération de leads via réseaux sociaux
- **Social Listening Sales** : écoute sociale pour opportunités de vente
- **Prospecting** : prospection ciblée sur les réseaux (LinkedIn Sales Navigator, X)
- **Pipeline ML** : pipeline de vente boosté par ML
- **Engagement Scoring** : scoring d'engagement des prospects
- **Account-Based Marketing** : marketing basé sur les comptes cibles

### Métriques

- Social Selling Index (SSI) LinkedIn
- Conversion rate par canal social
- Pipeline value attribuée aux réseaux sociaux
