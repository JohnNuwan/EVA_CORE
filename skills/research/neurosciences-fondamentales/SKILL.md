# Compétence : Neurosciences Fondamentales

## Présentation

Compétence niveau ingénieur/docteur en neurosciences : neurone, potentiel d'action, synapse, neurotransmetteurs, systèmes sensoriels, neuroanatomie.

---

## 1. Le Neurone

### Structure
```
Dendrites → Soma → Cône axonal → Axone → Terminaisons synaptiques
```
### Types
- **Pyramidal** (cortex, glutamate), **Purkinje** (cervelet, GABA)
- **Épineux moyens** (striatum, GABA), **Motoneurones** (ACh)

---

## 2. Potentiel d'Action

### Potentiel de repos (-70 mV)
| Ion | Intra (mM) | Extra (mM) | E_ion (mV) |
|---|---|---|---|
| K⁺ | 140 | 5 | -89 |
| Na⁺ | 15 | 150 | +62 |
| Ca²⁺ | 0.0001 | 2 | +123 |
| Cl⁻ | 10 | 110 | -63 |

### Phases
```
Dépolarisation (Nav ouverts) → Repolarisation (Kv ouverts)
→ Hyperpolarisation (Kv lents) → Période réfractaire (absolue/relative)
```

### Conduction saltatoire
- Myéline (oligodendrocytes SNC, Schwann SNP)
- Nœuds de Ranvier (Nav1.6 concentrés)
- Vitesse : 50-70 m/s (myélinisé) vs 0.5-2 m/s (non myélinisé)

---

## 3. Synapse

### Cycle vésiculaire
```
PA → Cav → Ca²⁺ → fusion SNARE (synaptobrevine/syntaxine/SNAP-25)
→ NT libéré → récepteur → PSP → recyclage (endocytose clathrine)
```

### PPSE / PPSI
| Type | ΔV | Canal |
|---|---|---|
| **PPSE** | +5 mV | AMPA/Na⁺ |
| **PPSI** | -5 mV | GABA_A/Cl⁻ |
| **NMDA** | +10 mV | Ca²⁺ (voltage-dépendant) |

---

## 4. Neurotransmetteurs

| NT | Synthèse | Récepteurs | Inactivation | Fonction |
|---|---|---|---|---|
| **Glutamate** | Glutaminase | AMPA, NMDA, mGluR | EAAT | Excitateur |
| **GABA** | GAD | GABA_A (Cl⁻), GABA_B | GAT | Inhibiteur |
| **ACh** | ChAT | nAChR, mAChR | AChE | NMJ, attention |
| **Dopamine** | TH → DOPA → DA | D1-D5 (GPCR) | DAT, COMT | Récompense |
| **Sérotonine** | TPH → 5-HTP | 5-HT₁-₇ | SERT | Humeur |

---

## 5. Systèmes Sensoriels

### Vision
- Bâtonnets (scotopique), cônes L/M/S (photopique)
- Phototransduction : rhodopsine → transducine → PDE → GMPc↓ → Na⁺ canaux ↓
- Voie : rétine → LGN → V1

### Audition
- Cochlée : IHC (3500) + OHC (12000), tonotopie
- Stéréocils → MET → K⁺ → dépolarisation
- Voie : CN → SOC → IC → MGB → A1

### Somesthésie
- **Toucher** : Aβ → colonnes dorsales → lemnisque médian → S1
- **Douleur** : Aδ (rapide, spinothalamique), C (lente, cingulaire)

---

## 6. Plasticité

### LTP (100 Hz)
```
Ca²⁺ via NMDA → CaMKII → ↑ AMPA + ↑ conductance → LTP
CaMKII auto-P T286 : mémoire moléculaire
```

### LTD (1 Hz)
```
Ca²⁺ faible → phosphatases → déP AMPA → internalisation
```
