# Compétence : Métabolisme et Bioénergétique

## Présentation

Compétence niveau ingénieur/docteur en métabolisme : glycolyse, TCA, phosphorylation oxydative, β-oxydation, photosynthèse.

## 2. Glycolyse

### Phase d'investissement (2 ATP)
```
Glucose → G6P → F6P → F1,6BP → DHAP + G3P (×2)
```
### Phase de récupération (4 ATP + 2 NADH)
```
G3P → 1,3-BPG → 3-PG → 2-PG → PEP → Pyruvate
```
**Bilan** : Glucose + 2 NAD⁺ + 2 ADP → 2 Pyruvate + 2 NADH + 2 ATP

### Régulation PFK-1 : ↑ AMP, F-2,6-BP ; ↓ ATP, Citrate

---

## 3. Cycle TCA (Krebs)

### 8 réactions dans matrice mitochondriale
```
Acétyl-CoA + OAA → Citrate → Isocitrate → α-KG → Succinyl-CoA → Succinate → Fumarate → Malate → OAA
```
**Bilan** : 3 NADH + FADH₂ + GTP + 2 CO₂
**Régulation** : ↑ Ca²⁺, ADP ; ↓ NADH, ATP

---

## 4. Chaîne Respiratoire

| Complexe | Pompage H⁺ | Inhibiteur |
|---|---|---|
| **CI : NADH → Q** | 4 H⁺ | Roténone |
| **CII : Succinate → Q** | 0 | — |
| **CIII : Q → Cyt c** | 4 H⁺ | Antimycine A |
| **CIV : Cyt c → O₂** | 2 H⁺ | CN⁻, CO |
| **CV : ATP synthase** | — | Oligomycine |

### Force proton-motrice : Δψ (~180 mV) + ΔpH → ATP
**Rapport P/O** : ~2.5 (NADH), ~1.5 (FADH₂)

---

## 5. β-Oxydation

### Palmitate (C16:0)
```
7 cycles → 8 Acétyl-CoA + 7 FADH₂ + 7 NADH
→ 106 ATP/palmitate
```
### Transport : Carnitine (CPT-1, inhibé par malonyl-CoA)

---

## 6. Photosynthèse

### Phase claire (thylakoïdes)
```
PSII (P680) : H₂O → O₂ + e⁻ → PQ → Cyt b6f → PC → PSI (P700) → NADP⁺
ATP synthase : photophosphorylation
```

### Cycle de Calvin (stroma)
```
3 CO₂ + 9 ATP + 6 NADPH → G3P
RuBisCO : CO₂ + RuBP → 2×PGA
Photorespiration : O₂ → 2-PG (perte)
Plantes C4 : évitent photorespiration
```
