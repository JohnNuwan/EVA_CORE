# Compétence : Biologie Cellulaire Fondamentale

## Présentation

Compétence niveau ingénieur/docteur en biologie cellulaire : membrane plasmique, organites, cytosquelette, trafic vésiculaire, matrice extracellulaire, jonctions.

---

## 1. Membrane Plasmique

### Structure bicouche
- **Phospholipides** : PC, PE, PS, PI, SM, cholestérol
- **Asymétrie** : feuillet externe (PC, SM, glycolipides) vs interne (PE, PS, PI)
- **Fluidité** : régulée par cholestérol, désaturases AG
- **Radeaux lipidiques** : domaines enrichis en SM + cholestérol

### Protéines membranaires
- **Intégrales** : TM α-hélice, β-barillet, ancres GPI/lipides
- **Périphériques** : électrostatiques, protéine-lipide
- **Transport** : canaux, uniport, symport, antiport, pompes ATPases

### Schéma
```
Glycoprotéines → Glycocalyx
Feuillet externe PC/SM/glycolipides
● Cholestérol
Feuillet interne PE/PS/PI
║ Protéines canal/transport
```

## 2. Organites

### Noyau
- Enveloppe nucléaire (NE), pores NPC, lamina (lamines A/B/C)
- Import NLS (PKKKRKV) → importine → Ran-GTP
- Compartiments : nucléole (ARNr), speckles (épissage), Cajal bodies

### RE (Réticulum Endoplasmique)
- **RE rugueux** : repliement, glycosylation N-liée, UPR (BiP, calnexine)
- **RE lisse** : lipides, Ca²⁺, détoxification (P450)

### Golgi
- Cis → médian → trans : glycosylation, tri (M6P → lysosomes)

### Lysosomes
- pH 5, V-ATPase, ~60 hydrolases (cathepsines, lipases)
- Autophagie : mTOR régule, autophagosome → lysosome

### Mitochondries
- Double membrane, cristae, espace inter-membranaire, matrice
- ADNmt (16.6 kb), fission/fusion (Drp1, Mfn1/2, OPA1)

---

## 3. Cytosquelette

### Microfilaments (actine)
- G-actine → F-actine (double hélice), extrémité + et -
- Profiline, cofiline, Arp2/3 (branchement), formine (nucléation)
- Myosines I, II, V, VI

### Microtubules
- α/β-tubuline → protofilaments (13) → microtubule (25 nm)
- Instabilité dynamique, GTP-cap, catastrophe/sauvetage
- Kinésine (+), dynéine (-), MAPs (Tau, MAP2)

### Filaments intermédiaires
- Kératines, vimentine, desmine, GFAP, neurofilaments, lamines

---

## 4. Trafic Vésiculaire

### Vésicules
| Type | Manteau | Direction |
|---|---|---|
| **COPII** | Sar1, Sec23/24/13/31 | RE → Golgi |
| **COPI** | ARF1 | Golgi → RE |
| **Clathrine** | AP2, dynamine | TGN/PM → endosomes |

### Fusion SNARE
```
v-SNARE (VAMP) + t-SNARE (syntaxine, SNAP-25) → complexe 4 hélices → fusion
Rab GTPases : marqueurs d'identité
```

---

## 5. Matrice Extracellulaire

| Composant | Rôle |
|---|---|
| **Collagène** (I-XXVIII) | Structure, force |
| **Élastine** | Fibres élastiques (cross-link lysyloxydase) |
| **Protéoglycanes** | GAGs + protéine core |
| **Fibronectine** (RGD) | Intégrines → adhésion |

### Jonctions
- **Serrees** : claudines, occludines, ZO-1 → barrière
- **Adhérentes** : cadhérines → caténines → actine
- **Desmosomes** : cadhérines → filaments intermédiaires
- **Gap** : connexines → connexons (hexamères) → canaux
