# Compétence : Neurobiologie Moléculaire

## Présentation

Compétence niveau ingénieur/docteur en neurobiologie moléculaire : canaux ioniques, récepteurs glutamatergiques et GABAergiques, plasticité, neurogénétique, pharmacologie.

## 2. Récepteurs Glutamate

### AMPA (GluA1-4)
- Na⁺/K⁺ ; Ca²⁺ si GluA2 absent
- PKA (S845), CaMKII (S831) → ↑ conductance

### NMDA (GluN1 + GluN2A-D)
- Glutamate + glycine + dépolarisation (chasse Mg²⁺) → Ca²⁺
- LTP (Ca²⁺ fort) ou LTD (Ca²⁺ faible)

### mGluR (métabotropes)
| Groupe | Effet |
|---|---|
| **I** (mGluR1/5) | Gαq → PLC → LTD |
| **II/III** | Gαi → ↓ cAMP → ↓ NT |

## 3. Plasticité Synaptique

### LTP (100 Hz)
```
Ca²⁺ → CaMKII → ↑ AMPA (insertion + conductance) → LTP
CaMKII auto-P T286 : mémoire moléculaire
CREB → BDNF, c-fos
```

### LTD (1 Hz)
```
Ca²⁺ faible → phosphatases → déP AMPA → internalisation
mGluR-LTD : PKC → traduction locale
```

### Structure
- Cofiline (dépolymérise actine), Arp2/3 (nucléation)
- Rho GTPases : Rac1 (croissance), RhoA (rétraction)
- PSD-95 : échafaudage, corrélé à force synaptique

---

## 4. Récepteurs GABA

### GABA-A (ionotrope, Cl⁻)
- Pentamère 2α:2β:1γ
- Sites : GABA, BZD (α1: sédatif, α2: anxiolytique), barbituriques

### GABA-B (métabotrope)
- GB1+GB2, Gαi → Kir3 → K⁺ → hyperpolarisation

## 5. Récepteurs Dopamine

| Type | Gα | Fonction |
|---|---|---|
| **D1** | Gαs | Mouvement (voie directe) |
| **D2** | Gαi | Mouvement (voie indirecte), auto-récepteur |

### Voies
- Nigrostriée : Parkinson
- Mésolimbique (VTA→NAc) : récompense
- Mésocorticale (VTA→PFC) : cognition
