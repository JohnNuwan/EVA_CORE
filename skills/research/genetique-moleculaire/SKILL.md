# Compétence : Génétique Moléculaire

## Présentation

Compétence niveau ingénieur/docteur en génétique moléculaire : ADN, réplication, transcription, traduction, code génétique, régulation, mutations.

---

## 1. Structure de l'ADN

### Double hélice (Watson & Crick, 1953)
- **Paires** : A=T (2 H), G≡C (3 H)
- **Pas** : 10.4 pb/tour, 3.4 nm
- **Antiparallèle** : 5'→3' et 3'→5'
- **Conformations** : B (standard), A (déshydraté), Z (gauche)

### Organisation
- **Nucléosome** : ADN 147 pb + octamère histones (H2A, H2B, H3, H4) ×2
- **Fibre 30 nm** + histone H1
- **TAD** : domaines topologiques (~1 Mb)
- **Télomères** : TTAGGG, shelterine
- **Centromères** : α-satellite, kinétochore

---

## 2. Réplication

### Fourche de réplication
```
Hélicase (déroule) → SSB (stabilise) → Topoisomérase (relâche torsion)
Primase → amorce ARN → ADN Pol III/δ/ε (élongation 5'→3')
Brin avancé : continu
Brin retardé : fragments Okazaki → ligase
```

| ADN Pol | Rôle |
|---|---|
| **Pol III** (E. coli) | Réplication principale |
| **Pol δ/ε** (eucaryote) | Brin retardé/avancé |
| **Pol γ** | ADNmt |
| **Télomérase** | TERT + ARN TER → allonge télomères |

## 3. Transcription

### ARN polymérases eucaryotes
| Pol | Produit | α-amanitine |
|---|---|---|
| **Pol I** | ARNr 28S, 18S, 5.8S | Résistante |
| **Pol II** | ARNm, snoARN | Très sensible |
| **Pol III** | ARNt, ARNr 5S | Sensible |

### Cycle Pol II
1. Initiation : TFIID (TBP→TATA) → TFIIA/B/F/E/H → PIC
2. Ouverture : TFIIH (XPB/XPD) → bulle ~18 nt
3. Élongation : P-TEFb → CTD phosphorylation
4. Terminaison : polyA signal (AAUAAA) → clivage → polyA

### Maturation ARNm
- **Cap** m⁷G (5'→5'), **Épissage** (spliceosome U1/U2/U4/U5/U6), **PolyA** (~200 A)
- **Épissage alternatif** : jusqu'à ~100k isoformes humain

## 4. Code Génétique et Traduction

### Code génétique
- 64 codons → 20 AA + 3 stop
- Dégénéré, universel, wobble (3ème base flexible)

### Traduction (ribosome)
```
Initiation : petit ribosome → AUG (Kozak) → grand ribosome
Élongation : EF-Tu/eEF1α → peptidyl transférase → EF-G/eEF2
Terminaison : eRF1/eRF3 → relargage
```

## 5. Régulation

### Niveaux
| Niveau | Temps | Mécanisme |
|---|---|---|
| **Chromatine** | min-h | Acétylation/méthylation histones |
| **Transcription** | min-h | Facteurs de transcription |
| **Post-transcription** | min | Épissage, stabilité ARNm |
| **Traduction** | s-min | miRNA, uORF |

### Code histone
| Modification | Effet |
|---|---|
| H3K4me3 | Activation |
| H3K9me3 | Répression (HP1) |
| H3K27me3 | Répression (Polycomb) |
| H3K9ac | Activation |

### Méthylation ADN
- CpG islands → promoteurs
- DNMT3a/b (de novo), DNMT1 (maintien)
- TET (déméthylation)
