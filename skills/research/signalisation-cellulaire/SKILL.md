# Compétence : Signalisation Cellulaire

## Présentation

Compétence niveau ingénieur/docteur en signalisation cellulaire : GPCR, RTK, JAK/STAT, Wnt, Notch, Hedgehog, TGF-β, PI3K/Akt/mTOR.

---

## 1. GPCR (7TM)

### Cycle
```
Ligand → Gα échange GDP→GTP → Gα+Gβγ libres → effecteurs
→ Hydrolyse GTP (RGS) → Réassociation
```

### Cascades
| Gα | Effecteur | 2nd messager |
|---|---|---|
| **Gαs** | AC (+) | cAMP → PKA |
| **Gαi** | AC (−) | cAMP ↓ |
| **Gαq** | PLCβ | IP₃, DAG, Ca²⁺ |
| **Gα12/13** | RhoGEF | RhoA, ROCK |

### Désensibilisation
GRK → phosphorylation → β-arrestine → internalisation

## 2. RTK → Ras-MAPK

```
Ligand → dimérisation → autophosphorylation
→ Grb2 → SOS → Ras-GTP → Raf → MEK → ERK → transcription
```
### Familles : EGFR, INSR, PDGFR, VEGFR, FGFR, Trk

---

## 3. JAK-STAT

```
Cytokine → récepteur → JAK → STAT → dimère → noyau → transcription
```
JAK1/2/3, TYK2 ; STAT1 (IFN), STAT3 (IL-6), STAT5 (GH)

---

## 4. Wnt

### Canonique (β-caténine)
```
OFF : APC/Axin/GSK3β → β-caténine dégradée
ON  : Wnt → Frizzled → Dvl → GSK3β inactivé → β-caténine stable
→ TCF/LEF → myc, cycline D
```
### Non canonique : Wnt/PCP (RhoA, JNK), Wnt/Ca²⁺ (PKC)

---

## 5. Notch

```
Delta/Jagged → Notch → clivage S2 (ADAM10) + S3 (γ-sécrétase)
→ NICD → CSL + Mastermind → HES, HEY
```

---

## 6. Hedgehog

```
SHH → Ptch + Smo → Gli-A (activateur) → cibles : PTCH1, Gli1, myc
OFF : Ptch inh. Smo → Gli-R (répresseur)
```

---

## 7. TGF-β / PI3K

### TGF-β
```
TβRII → TβRI → Smad2/3 + Smad4 → noyau → p21, p15
TAK1 → p38/JNK → EMT
```

### PI3K/Akt/mTOR
```
RTK → PI3K → PIP₃ → PDK1 → Akt → mTORC1 → synthèse protéique
                     → FOXO → gènes métaboliques
PTEN : antagoniste, suppresseur de tumeur
```
