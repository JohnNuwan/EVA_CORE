# Sources Fiables pour la Recherche Documentaire

Voici l'index des sources de référence à consulter en priorité selon le domaine d'expertise visé.

## 1. Intelligence Artificielle & Machine Learning
*   **arXiv** : `https://arxiv.org/` — Papiers de recherche originaux de dernière date.
*   **Hugging Face** : `https://huggingface.co/` — Modèles, datasets et documentation de l'écosystème open-source.
*   **Papers with Code** : `https://paperswithcode.com/` — Implémentations et codes sources liés aux publications de recherche.
*   **Meta AI / FAIR** : `https://ai.meta.com/research/` — Recherches sur les modèles du monde (JEPA, LLaMA).
*   **GitHub Search** : `https://github.com/` — Codes de référence réels et dépôts officiels des frameworks.

## 2. Automatisation Industrielle & Réseaux OT
*   **Siemens Industry Support** : `https://support.industry.siemens.com/` — Manuels techniques S7-1200/1500, SCL, OPC UA.
*   **OPC Foundation** : `https://opcfoundation.org/` — Spécifications officielles OPC UA et schémas XML NodeSet.
*   **Rockwell Automation** : `https://rockwellautomation.custhelp.com/` — Documentation Studio 5000 et langages ladder/SFC.
*   **Codesys Help** : `https://help.codesys.com/` — Manuels et guides d'implémentation de la norme CEI 61131-3.

## 3. Sécurité Industrielle & Cyber
*   **NVD / NIST** : `https://nvd.nist.gov/` — Base de vulnérabilités CVE et standards de sécurité.
*   **CISA ICS Advisories** : `https://www.cisa.gov/news-events/cybersecurity-advisories/` — Alertes de sécurité sur les systèmes de contrôle industriel.
